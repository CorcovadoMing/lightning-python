API Reference¶

Set up¶

create_session([name]) Create a lightning session.
use_session(session_id) Use the specified lightning session.
set_basic_auth(username, password) Set authenatication.
set_host(host) Set the host for a lightning server.
enable_ipython(**kwargs) Enable plotting in the iPython notebook.
disable_ipython() Disable plotting in the iPython notebook.

Plots¶

scatter(x, y[, color, label, size, alpha]) Plot two-dimensional data as points.
line(series[, index, color, label, size]) Plot one-dimensional series data as lines.
linestacked(series[, color, label, size]) Create a browsable array of line plots.
force(matrix[, color, label, size]) Create a force-directed network from a connectivity matrix.
graph(x, y, matrix[, color, label, size, ...]) Create a node-link graph from spatial points and their connectivity matrix.
graphbundled(x, y, matrix[, color, label, ...]) Create a node-link graph with bundled edges.
matrix(matrix[, colormap]) Visualize a dense matrix or table as a heat map.
adjacency(matrix[, label]) Visualize a sparse adjacency matrix.
map(regions, values[, colormap]) Create a chloropleth map of the world or united states.

Linked¶

scatterline(x, y, series[, color, label, ...]) Create a joint scatter / line plot.

Images¶

image(imagedata) Display an array as an image.
gallery(imagedata) Display a collection of arrays as browsable images with thumbnails.

3D¶

scatter3(x, y, z[, color, label, alpha, size]) Plot three-dimensional data as points.
volume(imagedata) Display a collection of images as a three-dimensional volume.

Streaming¶

scatterstreaming(x, y[, color, label, size]) Create a streaming scatter plot of x and y.
linestreaming(series[, index, color, label, ...]) Plot streaming one-dimensional series data as updating lines.
